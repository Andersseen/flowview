# Flowmark Astro Demo

This example shows how to integrate Flowmark with an Astro site. The `.flow`
templates are compiled to JavaScript render functions at build time using the
Flowmark CLI, and the Astro pages import and execute those render functions.

## Structure

```
examples/astro-demo/
├── src/
│   ├── generated/        # Generated by compile:templates (ignored by git)
│   ├── components/
│   ├── data/
│   ├── layouts/
│   ├── pages/
│   └── templates/
│       └── index.flow    # Flowmark template
└── scripts/
    └── compile-flow.mjs  # Compiles all .flow files to src/generated/
```

## Scripts

```sh
# Compile .flow templates
pnpm run compile:templates

# Start the dev server (compiles templates automatically)
pnpm run dev

# Build the static site (compiles templates automatically)
pnpm run build

# Preview the built site
pnpm run preview
```

From the repository root:

```sh
pnpm run demo              # start dev server
pnpm run build:demo        # build static site
pnpm run deploy:demo       # deploy to Cloudflare Pages
```

## How It Works

1. `scripts/compile-flow.mjs` runs `flowmark compile <template> --out <file>` for
every `.flow` file in `src/templates/`.
2. The generated files import `@flowmark/runtime` for HTML escaping helpers.
3. Astro pages import the generated `render(ctx)` function and call it with page
context.
4. The rendered HTML is injected into the Astro layout with
`<Fragment set:html={html} />`.

## Deploy to Cloudflare Pages

### Local deploy

1. Log in with Wrangler:
   ```sh
   pnpm exec wrangler login
   ```
2. Create a Cloudflare Pages project named `flowmark-demo` in the dashboard.
3. Set your account ID:
   ```sh
   export CLOUDFLARE_ACCOUNT_ID=your-account-id
   ```
4. Deploy:
   ```sh
   pnpm run deploy:demo
   ```

### GitHub Actions

The workflow `.github/workflows/deploy-demo.yml` deploys automatically on every
push to `main`. Add these secrets to your GitHub repository:

- `CLOUDFLARE_API_TOKEN` — create a token with **Cloudflare Pages:Edit** and
  **Zone:Read** permissions.
- `CLOUDFLARE_ACCOUNT_ID` — found on the right-hand side of the Cloudflare
  dashboard overview.
