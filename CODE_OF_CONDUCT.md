# Code of Conduct

flowview follows a simple rule: be respectful, direct, and constructive.

## Expected Behavior

- Use welcoming and inclusive language.
- Be kind when giving or receiving feedback.
- Focus criticism on ideas, code, and tradeoffs.
- Respect project scope and maintainer time.

## Unacceptable Behavior

- Harassment, threats, or personal attacks
- Insults, trolling, or sustained disruptive behavior
- Publishing private information without consent
- Discrimination or exclusionary behavior

## Enforcement

Maintainers may remove comments, close issues, reject contributions, or block
participants who violate this code of conduct.

If you need to report a concern, open a private security/contact channel with
the maintainers rather than escalating in public issue threads.
